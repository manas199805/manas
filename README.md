# Rice Quality Analyzer (BIS Standards)

This application analyzes rice grain quality based on image processing and Government (BIS) standards.

## Deployment Instructions
1. Upload `rice_app.py` to your GitHub repository.
2. Upload this `requirements.txt` file to the same repository.
3. Connect your repository to [Streamlit Community Cloud](https://share.streamlit.io/).

## Requirements
The `requirements.txt` file includes:
- streamlit
- opencv-python-headless
- numpy
- pandas
- scikit-image
- scipy
- Pillow
